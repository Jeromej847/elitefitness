# Elite Fitness Gym - Next.js Web Application

A modern web application for Elite Fitness Gym built with Next.js, React, and Tailwind CSS.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Multiple Training Programs** - Strength Training, Cardio Classes, and Yoga Sessions
- **Modern UI** - Built with Tailwind CSS for a sleek, contemporary look
- **Server-Side Rendering** - Fast performance with Next.js SSR capabilities

## Sections

- **Home** - Landing page with featured programs
- **Strength Training** - Information about strength training classes
- **Cardio Classes** - High-energy cardio workout options
- **Yoga Sessions** - Relaxing and rejuvenating yoga programs

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Jeromej847/elitefitness.git
cd elitefitness
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

## Building for Production

```bash
npm run build
npm start
```
